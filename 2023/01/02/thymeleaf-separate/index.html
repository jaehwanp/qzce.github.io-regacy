[thymeleaf] HTML 분리 (head, header, footer)

작성일 2023-01-02 In thymeleaf

개요

html에서 head, header, footer, nav 중복 쓰임을 줄이기 위함

환경설정

  • gradle
  • spring boot

build.gradle

구성

head.html

header.html

각 페이지 별로 header를 바꾸고 싶다면 파라미터를 설정할 수 있다.
<th:block th:fragment="headerFragment(headerParam)">

footer.html

index.html

html태그에 thymeleaf-layout-dialect 기능을 쓰기 위하여
xmlns:layout="http://www.ultraq.net.nz/thymeleaf/layout" 를 입력한다

head, header, footer를 index.html에 모아준다
