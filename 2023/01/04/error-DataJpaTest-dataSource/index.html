[Error] @DataJpaTest 외장 DB 에러

작성일 2023-01-04 In error

개요

에러

토이 프로젝트 중 에러가 남

마지막 줄을 보면 embedded database의 classpath가 맞지 않다고 나와있다.

나는 따로 설치한 mariaDB를 연동해서 쓰고있었다.

@AutoConfigureTestDatabase

AutoConfigureTestDatabase에서 따로 설정을 하지않으면 embedded Database로 테스트가 실행된다.

replace=AutoConfigureTestDatabase.NONE으로 값을 덮어 씌우면 설정해놓은 DB를 테스트에 사용할 수 있다.

해결

해당 어노테이션 추가하면 내장 DB가 아닌 내가 설정한 DataSource로 설정이 된다.

@AutoConfigureTestDatabase(replace = AutoConfigureTestDatabase.Replace.NONE)

@SpringBootTest
모든 설정을 가져와서 하기 때문에 이런거 필요없다.
